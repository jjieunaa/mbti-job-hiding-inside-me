## 수정할 점
1. 16개의 모든 MBTI별 특징이 아니라, 몇 개의 MBTI씩 묶어서 결과가 나오는 부분을 차차 수정해나갈 예정입니다.<br>
2. 앞의 4개 질문만 실제 질문이고, 뒤의 6개 질문은 가짜 질문입니다.<br>
이를 해결하기 위해 문제 수를 12개로 늘리고, 각 자리마다 많이 선택한 알파벳을 기준으로 결과를 낼 수 있도록 수정할 예정입니다.
